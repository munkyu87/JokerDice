# Voucher Ideas

현재 방향:

- 바우처 카드는 덱에 들어가는 `My Hand` 카드 형태
- 상점 전용으로 등장
- 사용 시 `discard`가 아니라 완전 제거
- 기본 가격은 `20G`
- 강한 메타 효과를 영구 적용

## 현재 구현됨

### Reserve Draw
- 효과: 매 Hand 시작 시 손패 +1 영구 증가
- 비고: 사용 후 완전 제거

### Negative Stamp
- 효과: 랜덤 비-네거티브 조커 1장을 네거티브로 변경
- 비고: 사용 후 완전 제거

### Interest Ledger
- 효과: 최대 이자 한도 +5 영구 증가
- 비고: 기본 25G -> 50G -> 75G 형태로 확장

### Boss Waiver
- 효과: 사용한 현재 Ante의 보스 효과 무시
- 비고: 보스 스테이지에서 안내 문구로 표시

## 미구현 아이디어

### Emergency Reroll
- 효과: 무료 리롤 2회 즉시 획득

### Purge Coupon
- 효과: 덱에서 카드 1장 영구 제거

### Lucky Stamp
- 효과: 이번 Hand 종료 시 골드 +4

### Spare Die
- 효과: 이번 Hand 동안 주사위 +1

### Static Breaker
- 효과: 보스의 조커 비활성화 효과만 무시

### Second Wind
- 효과: 이번 Hand 실패 시 한 번만 게임오버를 막고 Hand 1회 복구

### Mirror Ticket
- 효과: 손패의 다른 카드 1장을 복제본으로 생성

### Joker Polish
- 효과: 선택한 조커 1장의 효과 수치를 이번 Hand 동안 강화

## 나중에 검토할 확장안

- 보스 보상 전용 전설 바우처 추가
- 바우처 전용 프레임/아이콘 디자인
- 바우처 카드 전용 상점 슬롯
- 바우처 판매가 별도 책정
- 바우처 카드 전용 획득 사운드/연출
